## 1. 프로젝트 개요
### 프로젝트명: Community

### 목적
- 사용자 간의 자유로운 소통을 위한 커뮤니티 서비스 제공
- 게시글 및 댓글 기능 지원
- 다양한 사용자 인증 방법 제공
- 사용자의 권한을 세분화하여 서비스를 체계적으로 관리
<br><br><br>

## 2. 사용자 유형
### Admin(관리자)
- 매니저, 부매니저 지정 및 해제
- 회원 관리
- 게시판 관리
- 게시글 관리
- 댓글 관리
- 공지글 관리

### Manager(매니저) - 관리자에 의해 매니저로 지정된 User
- 부매니저 지정 및 해제
- 회원 관리
- 게시글 관리
- 댓글 관리
- 공지글 관리

### Sub Manager(부매니저) - 관리자/매니저에 의해 부매니저로 지정된 User
- 회원 관리
- 게시글 관리
- 댓글 관리
- 공지글 관리

### User(회원)
- 회원가입
- 프로필 관리
- 게시글 조회, 추천, 생성
- 댓글 조회, 추천, 생성
- 공지글 조회
<br><br><br>
