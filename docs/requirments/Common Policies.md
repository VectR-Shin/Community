### 활동 제한 정책
```
게시판 활동 정지 상태의 사용자는 다음 기능을 사용할 수 없다.

- 게시글 작성
- 게시글 수정
- 게시글 삭제
- 댓글 및 대댓글 작성
- 댓글 및 대댓글 수정
- 댓글 및 대댓글 삭제
- 추천/비추천
- 신고
```

<br>

### 인기글 정책
```
인기글은 게시 후 12시간 내에 받은 추천의 수가 30개 이상인 글 중에서 비추천의 수가 추천수의 30% 미만인 게시글이다.

인기글은 일반 사용자가 수정하거나 삭제할 수 없다.

인기글의 삭제는 매니저 이상의 관리자(ADMIN, MANAGER)만 가능하다.
```

<br>

### 인기댓글 정책
```
인기댓글은 등록 후 12시간 내에 받은 추천의 수가 10개 이상은 댓글 중에서 비추천의 수가 추천수의 30% 미만인 댓글이다.
```

<br>

### 프로필 공개 정책
```
자신의 프로필은 항상 조회 가능하다.

타인의 프로필은 공개 상태인 경우에만 조회 가능하다.
```

<br>

### 신고 정책
```
동일 사용자는 동일 게시글을 한 번만 신고할 수 있다.

동일 사용자는 동일 댓글을 한 번만 신고할 수 있다.

신고는 취소할 수 없다.
```

<br>

### 추천 정책
```
추천과 비추천은 동시에 지정할 수 없다.

추천 또는 비추천은 언제든지 취소할 수 있다.
```
