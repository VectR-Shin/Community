# Layouts
### Main Layout
```
목적
- 전역 네비게이션(Header) 기능 제공
- 게시글 통합 검색 기능 제공
- 게시판 목록 접근 기능 제공
- 사용자 인증 상태에 따른 사용자 기능 제공 (내 프로필, 로그인, 로그아웃)

적용 대상
- 추가 정보 입력 페이지 이외의 모든 페이지

공통 구성
- Header
  - 로고
  - 통합 검색
  - 게시판 목록 버튼
  - 로그인 상태에 따른 UI
    - 로그인: 내 프로필 버튼, 로그아웃 버튼
    - 비로그인: 로그인 버튼

공통 필요 데이터
- 로그인한 사용자의 프로필 정보

공통 사용자 동작
- 메인 페이지 이동 (로고 클릭)
- 게시판 목록 페이지 이동
- 게시글 통합 검색
- 인증 상태에 기반 동작
  - 로그인 사용자
    - 내 프로필 페이지 이동
    - 로그아웃
  - 로그아웃 사용자
    - 로그인 (GET /oauth2/authorization/keycloak)

공통 호출 API
- 로그인한 사용자 정보 조회
  - GET /users/me

비고
- Header 렌더링 시 최초 1회 호출
```

# Pages
### 추가 정보 입력 페이지 (Onboarding)
```
목적
- 처음 접속한 사용자의 세부 정보 입력을 위한 페이지

접근 대상
- 최초로 로그인한 온보딩 미완료 USER

레이아웃
- Onboarding Layout

필요 데이터
(없음)

사용자 동작
- 닉네임 입력 [필수]
- 가입 완료

호출 API
- 사용자 세부 정보(닉네임 등) 제출
  - POST /users/onboarding
```

<br>

### 메인 페이지
```
목적
- 사이트의 메인 페이지
- 전체 인기글 목록을 제공
- 제목 혹은 내용 기준으로 전체 인기글을 검색하는 기능 제공

접근 대상
- 모든 사용자

레이아웃
- Main Layout

필요 데이터
- 인기글 목록

사용자 동작
- 인기글 목록 조회
- 인기글 세부 페이지 이동
- 제목 기준으로 인기글 검색
- 내용 기준으로 인기글 검색

호출 API
- 전체 인기글 목록 조회
  - GET /posts/popular?page={pageNumber}
- 전체 인기글 검색 (제목/내용 필터 포함)
  - GET /posts/popular?keyword={keyword}&searchType={TITLE | CONTENT}&page={pageNumber}

비고
- 인기글 목록은 20개씩 페이지네이션 되어서 제공한다.
- 인기글 선정 기준은 운영 정책에 따라 변경될 수 있다. 요구사항 문서를 참고
```

### 게시판 페이지
```
목적
- 현재 게시판의 게시글 목록을 제공
- 현재 게시판의 인기글 목록을 제공
- 현재 게시판의 공지글 목록을 제공
- 현재 게시판의 게시글을 대상으로 하는 검색 기능 제공

접근 대상
- 모든 사용자

레이아웃
- Main Layout

필요 데이터
- 현재 게시판의 게시글 목록
- 현재 게시판의 인기글 목록
- 현재 게시판의 공지글 목록

사용자 동작
- 게시판 탐색
  - 현재 게시판의 게시글 / 인기글 / 공지글 조회
- 게시판 검색
  - 현재 게시판의 게시글 / 인기글 / 공지글 대상으로 제목 혹은 내용 기반 검색
- 게시글 상호작용
  - 게시글 상세 이동
  - 게시글 작성 이동 (로그인)

호출 API
- 현재 게시판의 게시글 목록 조회
  - GET /posts?boardId={boardId}&page={pageNumber}
- 현재 게시판의 인기글 목록 조회
  - GET /posts/popular?boardId={boardId}&page={pageNumber}
- 현재 게시판의 공지글 목록 조회
  - GET /notices?boardId={boardId}&page=1
- 현재 게시판의 게시글 검색
  - GET /posts?boardId={boardId}&keyword={keyword}&searchType={TITLE | CONTENT}
- 현재 게시판의 인기글 검색
  - GET /posts/popular?boardId={boardId}&keyword={keyword}&searchType={TITLE | CONTENT}

비고
- 게시글 목록과 인기글 목록은 20개씩 페이지네이션해서 제공한다.
- 공지글은 최근 20개 목록만을 제공한다.
- 모든 글은 최신순으로 제공한다.
```
