# Page Requirements
<br>

### 추가 정보 입력 페이지 (Onboarding)
```
목적
- 처음 접속한 사용자의 세부 정보 입력을 위한 페이지

접근 대상
- 최초로 로그인한 온보딩 미완료 USER

필요 데이터
(없음)

사용자 동작
- 닉네임 입력 [필수]
- 가입 완료

호출 API
- POST /users/onboarding
```

<br>

### 메인 페이지
```
목적
- 사이트의 메인 페이지
- 간단한 사용자 정보와 게시판 목록, 인기글 리스트를 제공

접근 대상
- ADMIN
- MANAGER
- SUB_MANAGER
- USER

필요 데이터
- 사용자 닉네임
- 모든 게시판 ID
- 모든 게시판명
- 

사용자 동작
-

호출 API
-

비고
- 인기글은 게시판과 무관하게 상위 10개를 제공
```
