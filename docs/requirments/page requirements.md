# Layouts
### Main Layout
```
목적
- 전역 네비게이션(Header) 기능 제공
- 게시글 통합 검색 기능 제공
- 게시판 목록 접근 기능 제공
- 사용자 인증 상태에 따른 사용자 기능 제공 (내 프로필, 로그인, 로그아웃)

적용 대상
- 추가 정보 입력 페이지 이외의 모든 페이지

공통 구성
- Header
  - 로고
  - 통합 검색
  - 게시판 목록 버튼
  - 로그인 상태에 따른 UI
    - 로그인: 내 프로필 버튼, 로그아웃 버튼
    - 비로그인: 로그인 버튼

공통 필요 데이터
- 로그인한 사용자의 프로필 정보

공통 사용자 동작
- 메인 페이지 이동 (로고 클릭)
- 게시판 목록 페이지 이동
- 통합 검색
  - 검색어 입력
  - 검색 대상 지정
  - 통합 검색 결과 페이지 이동
- 인증 상태에 기반 동작
  - 로그인 사용자
    - 내 프로필 페이지 이동
    - 로그아웃
  - 로그아웃 사용자
    - 로그인 (GET /oauth2/authorization/keycloak)

공통 호출 API
- 로그인한 사용자 정보 조회
  - GET /users/me

비고
- Header 렌더링 시 최초 1회 호출
```

### 게시판 목록 페이지
```
목적
- 전체 게시판 목록을 카테고리별로 묶어서 한 번에 제공
- 선택한 카테고리에 해당하는 게시판 목록을 제공
- 게시판 이름 기반의 게시판 검색 기능 제공

접근 대상
- 모든 사용자

레이아웃
- Main Layout

필요 데이터
- 게시판 목록

사용자 동작
- 게시판 탐색
  - 전체 게시판 목록 조회 (카테고리별로 grouping)
  - 특정 카테고리를 선택하여 게시판 필터링
- 게시판 검색
  - 게시판 이름 기반 검색

호출 API
- 전체 게시판 목록 조회
  - GET /boards
- 게시판 이름 기반의 게시판 검색 결과 조회
  - GET /boards?keyword={keyword}&page={pageNumber}

정책
- 검색 정책

비고
- 초기 화면에서는 전체 게시판 목록을 카테고리별로 묶어서 보여준다.
- 게시판 검색 결과는 20개씩 페이지네이션해서 제공한다.
- 카테고리 선택에 따른 게시판 목록은 GET /boards로 조회한 데이터를 프론트엔드에서 필터링하여 제공한다.
```

### 게시판 페이지
```
목적
- 게시글/공지글 작성 페이지 이동 기능 제공
- 게시글/공지글 상세 페이지 이동 기능 제공
- 현재 게시판의 게시글, 인기글, 공지글을 탭 형태로 제공
- 현재 게시판의 게시글, 인기글, 공지글 검색 기능을 제공
- 게시판 즐겨찾기 등록/삭제 기능 제공

접근 대상
- 모든 사용자

레이아웃
- Main Layout

필요 데이터
- 현재 게시판의 게시글 목록
- 현재 게시판의 인기글 목록
- 현재 게시판의 공지글 목록
- 현재 게시판의 즐겨찾기 등록 여부

사용자 동작
- 게시판 즐겨찾기
  - 현재 게시판 즐겨찾기 등록
  - 현재 게시판 즐겨찾기 해제
- 게시판 탐색
  - 게시글 / 인기글 / 공지글 탭 전환
  - 선택한 탭의 목록 조회
- 게시판 검색
  - 현재 게시판 탭(게시글 / 인기글 / 공지글)을 대상으로 제목 혹은 내용 기반 검색
- 게시글 상호작용
  - 게시글 상세 이동
  - 게시글 작성 이동
  - 공지글 작성 이동
  - 게시판 관리 페이지 이동

호출 API
- 현재 게시판 즐겨찾기 등록
  - POST /boards/{boardId}/favorite
- 현재 게시판 즐겨찾기 해제
  - DELETE /boards/{boardId}/favorite
- 현재 게시판의 게시글 목록 조회
  - GET /boards/{boardId}/posts?page={pageNumber}
- 현재 게시판의 인기글 목록 조회
  - GET /boards/{boardId}/posts/popular?page={pageNumber}
- 현재 게시판의 공지글 목록 조회
    GET /boards/{boardId}/notices?page={pageNumber}
- 현재 게시판의 게시글 탭 검색
    GET /boards/{boardId}/posts?keyword={keyword}&searchType={TITLE | CONTENT}&page={pageNumber}
- 현재 게시판의 인기글 탭 검색
  - GET /boards/{boardId}/posts/popular?keyword={keyword}&searchType={TITLE | CONTENT}&page={pageNumber}
- 현재 게시판의 공지글 탭 검색
  - GET /boards/{boardId}/notices?keyword={keyword}&searchType={TITLE | CONTENT}&page={pageNumber}

정책
- 활동 제한 정책
- 검색 정책
- 목록 조회 정책
- 게시판 정책
- 게시글 정책
- 공지글 정책

비고
- 게시글, 인기글, 공지글은 20개씩 페이지네이션해서 제공한다.
- 게시글 탭 상단에는 현재 게시판의 공지글 최대 5개를 함께 표시한다.
- 상단의 공지글 '더보기' 버튼을 눌러 공지글 탭으로 이동할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 에게만 관리 버튼을 표시한다.
- 관리 버튼을 클릭하면 현재 게시판의 식별자를 포함하여 게시판 관리 페이지로 이동한다.
```

  - 댓글 및 대댓글 관리 페이지 이동

  - DELETE /posts/{postId}/reactions

  - POST /posts/{postId}/reports
- 게시글의 댓글 목록 조회
  - GET /posts/{postId}/comments?page={pageNumber}

- 댓글 및 대댓글 삭제
  - DELETE /comments/{commentId}
- 댓글 및 대댓글 추천/비추천
  - POST /comments/{commentId}/reactions
- 댓글 및 대댓글 추천/비추천 취소

- 신고 정책

- 작성자 이름을 클릭했을 때, 작성자가 본인이라면 내 프로필 페이지로 이동하고, 타인이라면 타인의 프로필 페이지로 이동한다.
- 인기댓글은 추천이 가장 많은 3개만 제공한다.
- 게시글 및 댓글 신고는 신고 모달을 통해 처리한다.
- ADMIN, MANAGER, SUB_MANAGER 에게만 관리 버튼을 표시한다.
- 관리 버튼을 클릭하면 선택한 게시글 혹은 댓글의 식별자를 포함하여 게시글 혹은 댓글 관리 페이지로 이동한다.
