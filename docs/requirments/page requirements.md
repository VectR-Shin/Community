# Layouts
### Main Layout
```
적용 대상
- 추가 정보 입력 페이지 이외의 모든 페이지

공통 구성
- Header
  - 로고
  - 통합 검색
  - 게시판 목록 버튼
  - 로그인 상태에 따른 UI
    - 로그인: 내 프로필 버튼, 로그아웃 버튼
    - 비로그인: 로그인 버튼

공통 필요 데이터
- 로그인한 사용자의 프로필 정보

공통 사용자 동작
- 메인 페이지 이동
- 게시판 목록 페이지 이동
- 게시글 통합 검색
- 내 프로필 페이지 이동 (로그인 시)
- 로그인 (비로그인 시)
- 로그아웃 (로그인 시)
```

<br>

# Pages
### 추가 정보 입력 페이지 (Onboarding)
```
목적
- 처음 접속한 사용자의 세부 정보 입력을 위한 페이지

접근 대상
- 최초로 로그인한 온보딩 미완료 USER

레이아웃
- Onboarding Layout

필요 데이터
(없음)

사용자 동작
- 닉네임 입력 [필수]
- 가입 완료

호출 API
- POST /users/onboarding
```

<br>

### 메인 페이지
```
목적
- 사이트의 메인 페이지
- 간단한 사용자 정보와 게시판 목록, 인기글 리스트를 제공

접근 대상
- 모든 사용자

레이아웃
- Main Layout

필요 데이터
- 사용자 프로필 정보 (로그인한 경우)
- 게시판 목록
- 인기글 목록 (상위 20개)

사용자 동작
- 게시판 선택
- 인기글 선택
- 게시글 통합 검색
- 프로필 페이지 이동
- 로그인 페이지 이동 (GET /oauth2/authorization/keycloak)
- 로그아웃

호출 API
- GET /users/profile
- GET /boards/list
- GET /posts?page=1&sort=recommend

비고
- 인기글은 게시판과 무관하게 가장 추천이 많은 상위 20개를 제공
```
