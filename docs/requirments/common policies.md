### 운영 정책
```
- ADMIN 은 USER 이상의 사용자를 MANAGER, SUB_MANAGER 로 지정할 수 있다.
- ADMIN, MANAGER 는 USER 사용자를 SUB_MANAGER 로 지정할 수 있다.
- ADMIN 은 공지글 관리 기능을 사용할 수 있다.
- ADMIN 은 공지글 관리 기능을 통해 공지글을 삭제 및 복구할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 게시판 관리 기능을 사용할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 게시판 관리 기능을 통해 게시판을 숨김 처리 및 숨김 해제할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 게시글 관리 기능을 사용할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 게시글 관리 기능을 통해 게시글(인기글 포함)을 삭제 및 복구할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 댓글 관리 기능을 사용할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 댓글 관리 기능을 통해 댓글을 삭제 및 복구할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 사용자 관리 기능을 사용할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER 는 사용자 관리 기능을 통해 회원을 이용 정지 및 이용 정지 해제할 수 있다.
- ADMIN, MANAGER, SUB_MANAGER는 신고를 처리할 수 있다.
```

<br>

### 활동 제한 정책
```
- 게시판 활동 정지 상태의 사용자는 다음 기능을 사용할 수 없다.
  - 게시글 작성
  - 게시글 수정
  - 게시글 삭제
  - 댓글 및 대댓글 작성
  - 댓글 및 대댓글 수정
  - 댓글 및 대댓글 삭제
  - 추천/비추천
  - 신고

- 게시판 활동 정지 상태의 사용자는 읽기 기능을 제외한 모든 커뮤니티 활동을 수행할 수 없다.
```

<br>

### 검색 정책
```
- 검색 결과는 키워드와 완전 일치하는 결과를 우선적으로 제공하며, 이후 부분 일치 결과를 제공한다.
- 단, ID 기반의 모든 검색은 완전 일치하는 결과만을 제공한다.
```

<br>

### 목록 조회 정책
```
- 모든 글은 최신순으로 제공한다.
```

<br>

### 게시판 정책
```
- 게시판 상세 정보 조회는 ADMIN 만 가능하다.
- 게시판 즐겨찾기 등록/해제/조회는 로그인한 사용자만 가능하다.
```

<br>

### 게시글 정책
```
- 게시글 작성은 로그인한 사용자만 가능하다.
- 게시글 수정은 로그인한 게시글 작성자만 가능하다.
- 게시글 삭제는 로그인한 게시글 작성자만 가능하다.
- 게시글 수정 시 '(수정됨)' 표시를 추가한다.
- 게시글 삭제 시 사용자 확인 절차를 제공한다.
```

<br>

### 인기글 정책
```
- 인기글은 추천의 수가 30개 이상인 글 중에서 비추천의 수가 추천 수의 30% 미만인 게시글이다.
- 인기글은 일반 사용자가 수정하거나 삭제할 수 없다.
```

<br>

### 공지글 정책
```
- 공지글은 ADMIN, MANAGER, SUB_MANAGER만 작성할 수 있다.
- 공지글은 작성자만 수정할 수 있다.
- 공지글은 작성자만 삭제할 수 있다.
- 공지글은 추천/비추천 대상이 아니다.
- 공지글 수정 시 '(수정됨)' 표시를 추가한다.
- 공지글 삭제 시 사용자 확인 절차를 제공한다.
```

<br>

### 댓글 및 대댓글 정책
```
- 댓글 및 대댓글은 로그인한 사용자만 작성할 수 있다.
- 댓글 및 대댓글은 로그인한 댓글 작성자만 수정할 수 있다.
- 댓글 및 대댓글을 로그인한 댓글 작성자만 삭제할 수 있다.
- 댓글은 10개씩 페이지네이션하여 제공된다.
- 댓글 아래의 '대댓글 조회' 버튼을 누르면 대댓글을 추가로 제공한다.
- 대댓글은 1단계까지만 허용한다.
- 댓글 및 대댓글 수정 시 '(수정됨)' 표시를 추가한다.
- 댓글 및 대댓글 삭제 시 사용자 확인 절차를 제공한다.
```

<br>

### 인기댓글 정책
```
- 인기댓글은 추천 수가 10개 이상이면서 비추천의 수가 추천 수의 30% 미만인 댓글이다.
- 위 조건을 만족하는 댓글 중 추천이 가장 많은 3개만 제공한다.
- 인기댓글은 게시글 및 공지글의 첫 번째 댓글 페이지 최상단에 표시한다.
```

<br>

### 추천 정책
```
- 추천/비추천 지정 및 취소는 로그인한 사용자만 가능하다.
- 추천과 비추천은 동시에 지정할 수 없다.
- 추천 또는 비추천은 언제든지 취소할 수 있다.
```

<br>

### 프로필 공개 정책
```
- 자신의 프로필은 항상 조회 가능하다.
- 타인의 프로필은 공개 상태인 경우에만 조회 가능하다.
```

<br>

### 신고 정책
```
- 신고는 로그인한 사용자만 가능하다.
- 동일 사용자는 동일 게시글을 한 번만 신고할 수 있다.
- 동일 사용자는 동일 댓글을 한 번만 신고할 수 있다.
- 신고는 취소할 수 없다.
- 신고 처리는 운영 정책을 따른다.
```
